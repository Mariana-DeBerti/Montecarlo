Simulación de Monte Carlo

Mariana De Berti - 2026

¿Qué es la simulacion de Monte Carlo?

Es una técnica estadística que predice los posibles resultados de un evento incierto mediante el uso de muestreo aleatorio

Al simular miles de escenarios con variables aleatorias, permite evaluar riesgos y posibilidades

¿En qué consiste?

  • Definir un modelo matemático que representa el sistema que se quiere analizar
  • Generar números aleatorios para simular las variables de entrada del modelo
  • Ejecutar el modelo con los valores aleatorios y obtener resultados
  • Repetir el proces miles o millones de veces para obtener una distribución de resultados posibles

¿Para qué se utiliza?

  • Evaluar riesgos, incertidumbres en proyectos y decisiones
  • Analizar sistemas complejos y no lineales
  • Estimar probabilidades y valores esperados
  • Optimizar procesos y sistemas
